How to Play:

1. Insert Node: Add a new node to the end of the linked list by entering a value and clicking "Insert Node"

2. Delete Node: Remove a node from the list by entering its value and clicking "Delete Node"

3. Search Node: Find a node in the list by entering its value and clicking "Search Node"

4. Observe: Watch how the nodes are connected and how the list structure changes with each operation

5. Complete each objective in sequence to master the linked list operations

For Insert/Delete: Enter a value (e.g., 42)
For Position Operations: Enter position (0-based index)
For Reverse/Clear: No input needed
