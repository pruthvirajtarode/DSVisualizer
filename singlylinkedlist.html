Basic Structure

Address holding the next nodes

Singly Linked List Information

A singly linked list consists of nodes where each node contains data and a pointer to the next node. The head pointer points to the first node, and the last node's next pointer is null.

Key Characteristics of Singly Linked Lists

  • Dynamic Size: Can grow and shrink during program execution
  • Non-contiguous Memory: Nodes can be stored anywhere in memory
  • Single Direction: Can only traverse in one direction (forward)
  • Memory Efficiency: No wasted space for unused elements
  • No Random Access: Must traverse from head to access elements
  • Extra Memory: Requires additional space for pointers

Time Complexity

  • Access: O(n) - Must traverse from head to target node
  • Search: O(n) - Must check each node sequentially
  • Insertion at Head: O(1) - Constant time operation
  • Insertion at Tail: O(n) - Must traverse to end
  • Deletion at Head: O(1) - Constant time operation
  • Deletion at Tail: O(n) - Must traverse to end

Applications of Singly Linked Lists

  • Implementation of Stacks and Queues
  • Undo functionality in text editors
  • Browser history navigation
  • Polynomial arithmetic
  • Memory management in operating systems
  • Implementation of adjacency lists in graphs

Basic Singly Linked List Operations Pseudocode

// Node Structure
class Node:
    data: any
    next: Node

// Linked List Class
class LinkedList:
    head: Node

    // Insert at beginning
    function insertAtHead(data):
        newNode = new Node(data)
        newNode.next = head
        head = newNode

    // Insert at end
    function insertAtTail(data):
        newNode = new Node(data)
        if head is null:
            head = newNode
            return
        current = head
        while current.next is not null:
            current = current.next
        current.next = newNode

    // Delete from beginning
    function deleteAtHead():
        if head is null:
            return
        head = head.next

    // Search for a value
    function search(target):
        current = head
        while current is not null:
            if current.data == target:
                return true
            current = current.next
        return false

    // Traverse the list
    function traverse():
        current = head
        while current is not null:
            print(current.data)
            current = current.next
