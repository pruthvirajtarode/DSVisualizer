Hashing in Data Structures

Introduction to Hashing

Hashing is a technique that maps data of arbitrary size to fixed-size values. It uses a hash function to compute an index into an array of buckets or slots, from which the desired value can be found. This technique is widely used for implementing associative arrays, database indexing, caches, and sets.

Basic Hash Function

A hash function takes an input (key) and produces a fixed-size output (hash value) that serves as an index in the hash table.

Collision Resolution

When two different keys hash to the same index, a collision occurs. Various methods like chaining and open addressing are used to handle collisions.

Separate Chaining

In separate chaining, each bucket contains a linked list of elements that hash to that bucket. Multiple elements can be stored at the same index.

Key Characteristics

  • Constant-time average case operations
  • Efficient data retrieval
  • Space-efficient for storing large datasets
  • Perfect for implementing dictionaries and sets
  • Supports dynamic resizing for better performance

Time Complexity

  • Insert: O(1) average case
  • Search: O(1) average case
  • Delete: O(1) average case
  • Worst Case: O(n) when many collisions occur

Applications

  • Database indexing and fast data retrieval
  • Cryptography and password storage
  • Caching and memory management
  • Symbol tables in compilers
  • File systems and data deduplication

Implementation Example

class HashTable {
    private int[] table;
    private int size;
    
    HashTable(int size) {
        this.size = size;
        table = new int[size];
    }
    
    private int hash(int key) {
        return key % size;
    }
    
    void insert(int key) {
        int index = hash(key);
        while(table[index] != 0) {
            index = (index + 1) % size;
        }
        table[index] = key;
    }
    
    boolean search(int key) {
        int index = hash(key);
        while(table[index] != 0) {
            if(table[index] == key) {
                return true;
            }
            index = (index + 1) % size;
        }
        return false;
    }
}
