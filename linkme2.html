How to Play:

1. Insert at Start: Add a node at the beginning of the list

2. Insert at End: Add a node at the end of the list

3. Insert at Position: Add a node at a specific position

4. Delete from Start: Remove the first node

5. Delete from End: Remove the last node

6. Delete at Position: Remove a node from a specific position

7. Search: Find a value in the list

8. Update: Change a value at a specific position

9. Reverse: Reverse the entire list

10. Clear: Remove all nodes from the list

For Insert/Update: Enter value and optional position (e.g., "42" or "42,3")
For Delete/Search: Enter position (e.g., "3")
