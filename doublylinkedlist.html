Doubly Linked Lists in Data Structures

Doubly Linked List Structure

Doubly Linked List Information

A doubly linked list consists of nodes where each node contains data and two pointers: one pointing to the next node and another pointing to the previous node. This bidirectional linking enables efficient traversal in both directions.

Key Characteristics of Doubly Linked Lists

  • Bidirectional Traversal: Can move both forward and backward
  • Extra Memory: Requires additional space for previous pointers
  • Dynamic Size: Can grow and shrink during program execution
  • Non-contiguous Memory: Nodes can be stored anywhere in memory
  • Efficient Deletion: Can delete a node without traversing from head
  • Complex Implementation: More complex than singly linked lists

Time Complexity

  • Access: O(n) - Must traverse from head/tail to target node
  • Search: O(n) - Must check each node sequentially
  • Insertion at Head/Tail: O(1) - Constant time operation
  • Insertion at Middle: O(n) - Must traverse to position
  • Deletion at Head/Tail: O(1) - Constant time operation
  • Deletion at Middle: O(1) - Direct access with previous pointer

Applications of Doubly Linked Lists

  • Browser History: Forward and backward navigation
  • Text Editor: Undo and redo operations
  • Music Player: Previous and next track functionality
  • Image Viewer: Previous and next image navigation
  • Cache Implementation: LRU (Least Recently Used) cache
  • Navigation Systems: Bidirectional path finding

Basic Doubly Linked List Operations Pseudocode

// Node Structure
class Node:
    data: any
    next: Node
    prev: Node

// Doubly Linked List Class
class DoublyLinkedList:
    head: Node
    tail: Node

    // Insert at beginning
    function insertAtHead(data):
        newNode = new Node(data)
        if head is null:
            head = newNode
            tail = newNode
            return
        newNode.next = head
        head.prev = newNode
        head = newNode

    // Insert at end
    function insertAtTail(data):
        newNode = new Node(data)
        if tail is null:
            head = newNode
            tail = newNode
            return
        newNode.prev = tail
        tail.next = newNode
        tail = newNode

    // Insert at specific position
    function insertAtPosition(data, position):
        if position <= 0:
            insertAtHead(data)
            return
            
        newNode = new Node(data)
        current = head
        count = 0
        
        // Traverse to the position
        while current is not null and count < position - 1:
            current = current.next
            count += 1
            
        if current is null:
            insertAtTail(data)
            return
            
        // Insert the new node
        newNode.next = current.next
        newNode.prev = current
        if current.next is not null:
            current.next.prev = newNode
        current.next = newNode
        
        // Update tail if inserting at end
        if newNode.next is null:
            tail = newNode

    // Delete from beginning
    function deleteAtHead():
        if head is null:
            return
        if head == tail:
            head = null
            tail = null
            return
        head = head.next
        head.prev = null

    // Delete from end
    function deleteAtTail():
        if tail is null:
            return
        if head == tail:
            head = null
            tail = null
            return
        tail = tail.prev
        tail.next = null

    // Delete a specific node
    function deleteNode(node):
        if node is null:
            return
        if node == head:
            deleteAtHead()
            return
        if node == tail:
            deleteAtTail()
            return
        node.prev.next = node.next
        node.next.prev = node.prev

    // Search for a value
    function search(target):
        current = head
        while current is not null:
            if current.data == target:
                return true
            current = current.next
        return false

    // Traverse forward
    function traverseForward():
        current = head
        while current is not null:
            print(current.data)
            current = current.next

    // Traverse backward
    function traverseBackward():
        current = tail
        while current is not null:
            print(current.data)
            current = current.prev
