Stack Game

🎯 Current Objective:

Push the first item onto the stack.

Please enter a size between 1 and 9

How to Play:

  1. First, set your desired stack size
  2. Enter a number in the input field
  3. Use Push to add the number to the stack
  4. Use Pop to remove the top element
  5. Use Peek to see the top element without removing it
  6. Complete the current objective to win!

Stack Operations

  • Push: Adds an element to the top of the stack
  • Pop: Removes the top element from the stack
  • Peek: Views the top element without removing it

Stack Properties

  • LIFO (Last In First Out) structure
  • Only the top element is accessible
  • Operations are performed at one end only
