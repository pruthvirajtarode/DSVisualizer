Arrays

What is an Array?

An array is a linear data structure that stores elements of the same type in contiguous memory locations.

Key Operations

  • Access: O(1)
  • Search: O(n)
  • Insertion: O(n)
  • Deletion: O(n)

Linked Lists

What is a Linked List?

A linked list is a linear data structure where elements are stored in nodes, and each node points to the next node.

Key Operations

  • Access: O(n)
  • Search: O(n)
  • Insertion: O(1)
  • Deletion: O(1)

Trees

What is a Tree?

A tree is a hierarchical data structure consisting of nodes connected by edges.

Key Operations

  • Search: O(log n)
  • Insertion: O(log n)
  • Deletion: O(log n)

Stacks & Queues

What are Stacks & Queues?

Stacks and Queues are linear data structures that follow LIFO and FIFO principles respectively.

Key Operations

  • Push/Pop (Stack): O(1)
  • Enqueue/Dequeue (Queue): O(1)
