Arrays in Data Structures

Introduction to Arrays

An array is a linear data structure that stores elements of the same data type in contiguous memory locations. It provides direct access to elements using an index, making it one of the most fundamental and widely used data structures. Arrays can be one-dimensional (1D), two-dimensional (2D), or multi-dimensional, each serving different purposes in data organization and manipulation.

1D Array

A one-dimensional array is a linear collection of elements accessed using a single index. It's the simplest form of array where elements are stored in a single row.

2D Array

A two-dimensional array is a matrix-like structure where elements are arranged in rows and columns. It requires two indices to access elements: one for row and one for column.

3D Array

A three-dimensional array extends the concept to three dimensions, useful for representing 3D data like RGB images or 3D coordinates in space.

Key Characteristics of Arrays

  • Fixed Size: Arrays have a fixed size that must be declared at creation time
  • Contiguous Memory: Elements are stored in adjacent memory locations
  • Random Access: Direct access to any element using index in O(1) time
  • Homogeneous Elements: All elements must be of the same data type
  • Index-based: Elements are accessed using zero-based indexing
  • Cache-friendly: Due to contiguous memory allocation, arrays have good cache locality

Time Complexity

  • Access: O(1) - Direct access using index
  • Search: O(n) - Linear search through elements
  • Insertion: O(n) - Requires shifting elements
  • Deletion: O(n) - Requires shifting elements
  • Update: O(1) - Direct access and update

Applications of Arrays

  • Image Processing: Storing pixel data in 2D arrays
  • Matrix Operations: Used in mathematical computations
  • Database Records: Storing records in tabular format
  • Game Development: Representing game boards and grids
  • Audio Processing: Storing audio samples
  • Dynamic Programming: Storing intermediate results

Basic Array Operations Pseudocode

// Array Declaration
array = [1, 2, 3, 4, 5]

// Accessing Element
element = array[index]

// Inserting Element
function insert(array, index, element):
    for i = array.length - 1 to index:
        array[i + 1] = array[i]
    array[index] = element

// Deleting Element
function delete(array, index):
    for i = index to array.length - 2:
        array[i] = array[i + 1]
    array.length = array.length - 1

// Searching Element
function search(array, target):
    for i = 0 to array.length - 1:
        if array[i] == target:
            return i
    return -1
