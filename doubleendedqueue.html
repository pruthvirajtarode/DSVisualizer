How to Play:

  1. First, set your desired deque size (1-10)
  2. Enter a value in the input field
  3. Choose whether to insert at front or rear
  4. Click "Insert" to add the value to the selected end
  5. Click "Delete Front" to remove from front
  6. Click "Delete Rear" to remove from rear
  7. Click "Peek" to see the front element without removing it

Deque Operations:

  • Insert Front: Adds an element to the front of the deque
  • Insert Rear: Adds an element to the rear of the deque
  • Delete Front: Removes an element from the front of the deque
  • Delete Rear: Removes an element from the rear of the deque
  • Peek: Views the front element without removing it

Deque Properties:

  • Can function as both FIFO and LIFO
  • Elements can be added/removed from both ends
  • Dynamic size implementation
  • Efficient access to both ends
  • Versatile data structure for various applications
